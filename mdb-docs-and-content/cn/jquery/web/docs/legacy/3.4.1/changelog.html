Note 2: Because many of our users ask us if MDB Pro version contains also features of MDB Free if MDB Free has a higher number of the version than MDB Pro - we decided to change a numeration.

With this release we jump with MDB Pro to version 3.4.0. and version 3.3.3. for MDB Free. The previous numeration was a result of our internal development process. That's why please notice that:

- MDB Pro contains all the features of MDB Free (actually MDB Free is a part of MDB Pro)

- MDB Free doesn't contain a features and components of MDB Pro

20.01.2016

Important note:

MDB package is growing. Every month it offers more and more posibilities for our users but the same time it take more weight.

That's why we strongly encourage you to compile your MDB files to fit your project. For some projects you need particular components, for other - different components. It would be very rare situation that you need ALL the components in the single project.

Each MDB package (free and premium) contains sass files and javascript modules, which are just a small pieces of MDB framework. That's why it's very easy to compile an accurate and lightweight package which will fit your needs.
