You can easily apply the waves effect to every element in our Framework by adding a special class



class="waves-effect"



You can also change the color and intensity of the effect.


class="waves-effect"

class="waves-effect waves-light"

class="waves-effect waves-red"

class="waves-effect waves-yellow"

class="waves-effect waves-orange"

class="waves-effect waves-purple"

class="waves-effect waves-green"

class="waves-effect waves-teal"
