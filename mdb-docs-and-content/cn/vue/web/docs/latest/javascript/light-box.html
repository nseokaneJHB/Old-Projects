Bootstrap lightbox MDB Pro component

Bootstrap lightbox is a group of images combined in one responsive gallery. Elements are grouped in thumbnail grid, which can be displayed as a slideshow.

Basic example



Predefined image size

MDB LightBox uses the same image in 2 different sizes: thumbnail and large.

Such an approach let you increase the performance of your website.


Lightbox Gallery example:
