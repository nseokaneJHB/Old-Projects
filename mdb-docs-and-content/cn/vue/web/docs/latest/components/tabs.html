Bootstrap tabs & pills

Bootstrap tabs are components which separate content placed in the same wrapper, but in the separate pane. Only one pane can be displayed at the time.

Tabs available in MDB Vue share general markup and styles, from the base, .nav class to the active and disabled states. Their visual style depends on the props passed in; be sure to check out their features below.

For sake of reference, the examples include a tab-switching animation using <transition-group> and defined in snippets' <style> section.
