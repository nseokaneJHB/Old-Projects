Bootstrap Autocomplete

MDB Autocomplete is a component which predicts the words basing on the first few letters given by a user, while one is typing it. It takes in an Array full of suggestions.

Autocomplete value

Material input uses $emit inside component. To handle value changes use @input event instead of v-model.

Basic example MDB Pro component
