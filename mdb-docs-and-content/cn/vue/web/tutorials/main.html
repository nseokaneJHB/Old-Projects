What Vue is?

Vue is a progressive framework for building user interfaces. Unlike other monolithic frameworks, Vue is designed from the ground up to be incrementally adoptable. The core library is focused on the view layer only, and is easy to pick up and integrate with other libraries or existing projects. Vue is widely used worldwide among others by:

Vue is called by many a rising star among libraries due to the pace in which it has been gaining popularity.
