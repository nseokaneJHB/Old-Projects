Basic usage - styles

It requires two simple steps

Step 1: Add the class .modal-notify to the .modal-dialog div.

Step 2: Add the class .success, .info, .danger or .warning to the .modal-dialog div.


These classes will affect the following elements:

  1. .modal-header

  2. .badge

  3. .fa-4x - which stands for a large icon
