Bootstrap Simple Charts

Lightweight plugin to render simple, animated and retina optimized pie charts with customization option.

To start working with Simple Charts, please see "Getting Started" tab on this page.

Note: Simple Charts addon integration is available from version 4.8.10 (released 16.09.2019).

Options

You can pass these options to the initialize function to set a custom look and feel for the plugin.

Defaults options are following :

Name Type Custom Value Description
barColor color #ef1e25 The color of the curcular bar. You can either pass a valid css color string, or a function that takes the current percentage as a value and returns a valid css color string.
trackColor color #f2f2f2 The color of the track, or false to disable rendering.
scaleColor Color #dfe0e0 The color of the scale lines, false to disable rendering.
scaleLength Number 5 Length of the scale lines (reduces the radius of the chart).
lineCap Predefined string round Defines how the ending of the bar line looks like. Possible values are: butt, round and square.
lineWidth Number 3 Width of the chart line in px.
size Number 110 Size of the pie chart in px. It will always be a square.
rotate Number 0 Rotation of the complete chart in degrees.
animate object {} Object with time in milliseconds and boolean for an animation of the bar growing ({ duration: 1000, enabled: true }), or false to deactivate animations.
easing function defaultEasing Easing function or string with the name of a jQuery easing function

Callbacks

All callbacks will only be called if animate is not false.

Defaults options are following :

Callback(params, ...) Description
onStart(from, to) Is called at the start of any animation.
onStep(from, to, currentValue) Is called during animations providing the current value (the method is scoped to the context of th eplugin, so you can access the DOM element via this.el).
onStop(from, to) Is called at the end of any animation.
