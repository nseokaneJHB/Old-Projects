Angular SmoothScroll / Scrollbar - API

In this section you will find informations about required modules and available inputs, outputs, methods and events of Smooth Scroll.

Modules used

In order to speed up your application, you can choose to import only the modules you actually need, instead of importing the entire MDB Angular library. Remember that importing the entire library, and immediately afterwards a specific module, is bad practice, and can cause application errors. 

// For MDB Angular Pro
import { SmoothscrollModule, WavesModule } from 'ng-uikit-pro-standard'

Directives

PageScrollDirective

Selector: mdbPageScroll

Type: PageScrollDirective

Inputs

PageScrollDirective
Name Type Default Description Example
pageScrollHorizontal boolean false Whether the scroll should happen in vertical direction (false, default) or horizontal (true). [pageScrollHorizontal]="true"
pageScrollOffset number 0 Pixels to offset from the top of the element when scrolling to (positive value = scrolling will stop given pixels atop the target element). [pageScrollOffset]="100"
pageScrollDuration number 2000 Duration in milliseconds the whole scroll-animation should last. [pageScrollDuration]="1500"
pageScrollSpeed number 3000 Speed in Pixel/Second the animation should take. Only applied if no duration is set. [pageScrollSpeed]="1000"
pageScrollInterruptible boolean true Whether the scroll animation should stop if the user interferes with it (true) or not (false). [pageScrollInterruptible]="true"

Events

PageScrollDirective
Name Type Description Example
pageScrollFinish boolean Fired when the scroll-animation stops. Emits a boolean value which indicates whether the scroll animation finished successfully and reached its target (true) or not (false). Possible reasons for false: target not found or interrupted due to another scroll animation starting or user interaction. (pageScrollFinish)="onPageScrollFinish()"
