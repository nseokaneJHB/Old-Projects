Angular Mobile Cards

Angular Mobile Cards - Bootstrap 4 & Material Design

Cards are the basic building block for each application. Use them in the Material Design version on both platforms!

Basic example

Each mdb-card component must have his own separate wrapper - StackLayout.

Android
iOS
import {Component} from '@angular/core'; @Component({ selector: 'App', moduleId: module.id, templateUrl: './app.component.html', }) export class AppComponent { loremIpsumText = `Some quick example text to build on the panel title and make up the bulk of the panel's content.`; }
