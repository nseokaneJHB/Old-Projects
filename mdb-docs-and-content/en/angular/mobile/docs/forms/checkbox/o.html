Angular Mobile Checkbox

Angular Mobile Checkbox - Bootstrap 4 & Material Design

Angular Mobile Bootstrap checkbox is a component used to allow a user to make a multiple choice. It is broadly used in the forms and surveys.

Checkboxes are for selecting one or several options in a list, while radios are for selecting one option from many.

Checked state

Add a checked="true" input to pre-select checkbox when the page loads.

Disabled state

Add an isEnabled="false" to disable checkbox.

A disabled checkbox component is unusable and un-clickable.
