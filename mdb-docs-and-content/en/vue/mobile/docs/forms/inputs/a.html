Input - API

In this section you will find advanced information about the Input component. You will learn which modules are required in this component, what are the possibilities of configuring the component, and what events and methods you can use to work with it.

API Reference: Properties

Name Type Default Description Example
text String Sets an input example text <TextField text="Example text" />
hint String Sets an input placeholder <TextField hint="Placeholder" />
secure Boolean Enables password mode <TextField secure />
editable Boolean Disables input edition <TextField editable="false" />
maxLength Number Defines max lenght of an input value <TextField :maxLength="5" />

API Reference: Events

Name Description Example
@focus Listener for focus event <TextField @focus="handleFocus" />
@blur Listener for blur event <TextField @blur="handleBlur" />
