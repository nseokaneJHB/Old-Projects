Radio - API

In this section you will find advanced information about the Radio component. You will learn which modules are required in this component, what are the possibilities of configuring the component, and what events and methods you can use in working with it.

Import statement

Checkbox component has to be imported separately by using the following syntax:

API Reference: Properties

Name Type Default Description Example
checked Boolean false Pre-selects checkbox when the page loads. <Checkbox checked="true" />
fillColor String Defines checkbox fill color. <Checkbox fillColor="#ff3547" />
boxType String Defines checkbox box type. <Checkbox boxType="circle" />

API Reference: Methods

Name Parameters Description Example
@checkedChange $event Gets event data after each state change. Mainly used for getting checkbox value. <CheckBox :checked="checked" @checkedChange="checked = $event.value" />
