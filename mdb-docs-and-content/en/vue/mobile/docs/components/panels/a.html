Panels - API

In this section you will find advanced information about the Panels component. You will learn which modules are required in this component, what are the possibilities of configuring the component, and what events and methods you can use in working with it.

Import statement

import { MDBCard } from 'mdbvue-mobile'

API Reference: Properties

Name Type Default Description Example
img String '' Defines card's image <MDBCard img="https://(...)" />
title String '' Defines card's title <MDBCard title="Card title" />
text String '' Defines card's text <MDBCard text="Card text" />
imgOverlay String '' Defines card's image overlay <MDBCard imgOverlay="https://(...)" />
titleColor String 'black' Defines card's title color <MDBCard titleColor="blue" />
textColor String 'grey' Defines card's text color <MDBCard textColor="blue" />
align String 'left' Defines card's text alignment <MDBCard align="right" />
imgHeight String '150' Defines card's image height <MDBCard imgHeight="200" />
height String '200' Defines card's height <MDBCard height="200" />
