Badges - API

In this section you will find advanced information about the Badges component. You will learn which modules are required in this component, what are the possibilities of configuring the component, and what events and methods you can use in working with it.

Import statement

import { MDBBadge } from 'mdbvue-mobile'

API Reference: Properties

Name Type Default Description Example
mdbColor String '' Changes badge's color <MDBBadge mdbColor="info" />
pill Boolean false Changes badge into pill <MDBBadge pill />
icon String '' Fills badge content by icon <MDBBadge icon="user" />
far || regular Boolean false Defines Font Awesome icon type <MDBBadge icon="user" far />
fal || light Boolean false Defines Font Awesome icon type <MDBBadge icon="user" fal />
fab || brands Boolean false Defines Font Awesome icon type <MDBBadge icon="facebook" fab />
