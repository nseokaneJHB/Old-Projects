Button Group - API

In this section you will find advanced information about the Button Group component. You will learn which modules are required in this component, what are the possibilities of configuring the component, and what events and methods you can use in working with it.

Import statement

import { MDBBtn, MDBBtnGroup } from 'mdbvue-mobile'

API Reference: Properties

Name Type Default Description Example
align String 'left' Changes component's vertical alignment <MDBBtnGroup align="center">
vertical Boolean false Changes component flex direction <MDBBtnGroup vertical>
