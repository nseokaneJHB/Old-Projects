Dropdowns - API

In this section you will find advanced information about the Dropdowns component. You will learn which modules are required in this component, what are the possibilities of configuring the component, and what events and methods you can use in working with it.

Import statement

import { MDBBtn } from 'mdbvue-mobile'

API Reference: Properties

Name Type Default Description Example
mdbColor String '' Determines button color <MDBBtn mdbColor="primary" />
icon String '' Adds Font Awesome 5 icon inside button <MDBBtn mdbColor="primary" icon="user" />
far || regular Boolean false Defines Font Awesome icon type <MDBBtn icon="user" far />
fal || light Boolean false Defines Font Awesome icon type <MDBBtn icon="user" fal />
fab || brands Boolean false Defines Font Awesome icon type <MDBBtn icon="facebook" fab />
btnGroup Boolean false Determines button group class <MDBBtn btnGroup />
text String '' Determines button text <MDBBtn text="Button" />
textLeft Boolean false Determines if button text is placed on the left relative to icon <MDBBtn text="Button" textLeft icon="user" />
dropdown Boolean false Determines if button is a dropdown <MDBBtn text="Dropdown" dropdown />

API Reference: Methods

Name Parameters Description Example
@tap Evokes after tapping the button <MDBBtn @tap.native="handleTap" />
