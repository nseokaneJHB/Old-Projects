Collapse - API

In this section you will find advanced information about the Collapse component. You will learn which modules are required in this component, what are the possibilities of configuring the component, and what events and methods you can use in working with it.

Import statement

import { MDBCollapse } from 'mdbvue-mobile'

API Reference: Properties

Name Type Default Description Example
toggler String 'Toggle' Defines toggler text. <MDBCollapse toggler="Toggle button" >
togglerMDBColor String 'primary' Defines toggler color. <MDBCollapse togglerMDBColor="success" >
active Boolean false Defines collapse content default state. <MDBCollapse active >
banner Boolean false Defines if collapse toggler is button or label style. <MDBCollapse banner >
hideAll Boolean false Defines all collapse elements state after toggling. <MDBCollapse hideAll >
togglerColor String 'blue' Defines banner toggler color. <MDBCollapse togglerColor="red" >
togglerAlign String 'left' Defines banner toggler alignment. <MDBCollapse togglerAlign="right" >
speed String 'default' Defines collapse animation speed. Available options: default, slow, fast. <MDBCollapse speed="fast" >
