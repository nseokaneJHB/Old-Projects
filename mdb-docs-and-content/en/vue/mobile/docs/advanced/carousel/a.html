Carousel - API

In this section you will find advanced information about the Carousel component. You will learn which modules are required in this component, what are the possibilities of configuring the component, and what events and methods you can use in working with it.

Import statement

Carousel's components needs to be imported separately by using the following syntax:

Carousel Properties

Name Type Default Description Example
height String '' Defines carousel height. <Carousel height="100%">
width String '' Defines carousel width. <Carousel width="100%">
indicatorColor String '' Defines carousel indicator color. <Carousel indicatorColor="white">
showIndicator Boolean true Toggles indicators. <Carousel showIndicator="false">
android:indicatorAnimation String Defines indicator animation for android. Available options: none, color, scale, slide, worm, thin_worm, fill, drop, swap. <Carousel android:indicatorAnimation="swap">

CarouselItem Properties

Name Type Default Description Example
id String '' Defines slide id. <CarouselItem id="slide1">

For more configuration options please visit NativeScript Carousel plugin home page.
