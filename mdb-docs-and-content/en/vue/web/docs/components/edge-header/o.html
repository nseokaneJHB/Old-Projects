Vue Bootstrap Edge Header

Vue Bootstrap edge header is a full-width block located on the top of a page as a part of a background for a card. It is a component unique for Material Design, that can be often spotted in Google products and pages.

It gives additional depth and meaning to the project by making it seem like the card floats above its background.
