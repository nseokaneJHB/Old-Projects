Vue Slider - API

In this section you will find informations about required modules of slider component.

API Reference: Props

Name Type Default Description
min Number 0 Sets the lowest Slider's value
max Number 100 Sets the highest slider's value
value Number 50 Slider's value - useful for additional data binding and edge use-cases

API Reference: Events

Name Parameters Description Example
rangeChange value Once the slider's handle is set to a new value, the component emits the event with the new value parameter <mdb-range-input @rangeChange="change" />
