Vue Bootstrap Notification

Vue Bootstrap Notification - Bootstrap 4 & Material Design

Vue Bootstrap Notification is a set of components which displays a static information or one generated in response to particular user on-site behavior.

Basic examples

Alerts are available for any length of text, as well as an optional dismiss button. For proper styling, use one of the four required contextual classes (e.g., .alert-success). For inline dismissal, use the alerts jQuery plugin.
