Icons - API

In this section you will find advanced information about the Icon component. You will learn which modules are required in this component, what are the possibilities of configuring the component, and what events and methods you can use in working with it.

Import statement

import { mdbIcon } from "mdbvue";

API Reference: Icons Properties

The table below shows the configuration options of the Icon component.

Name Type Default Description Example
icon String Required! The Font Awesome icon name, which you want to use Icons list <mdb-icon icon="chevron" />
far Boolean false Sets regular icon category; alias: regular <mdb-icon far icon="user" />
fab Boolean false Sets brand icon category; alias: brand <mdb-icon fab icon="twitter" />
fal Boolean false Sets light icon category; alias: light <mdb-icon fal icon="user" />
fad Boolean false Sets duotone icon category; alias: light <mdb-icon fad icon="user" />
brand Boolean false Sets brand icon category; alias: fab <mdb-icon brand icon="twitter" />
regular Boolean false Sets regular icon category; alias: far <mdb-icon regular icon="twitter" />
light Boolean false Sets light icon category; alias: fal <mdb-icon light icon="twitter" />
duotone Boolean false Sets duotone icon category; alias: fad <mdb-icon duotone icon="twitter" />
size [Boolean, String] false Sets icon size <mdb-icon size="2x" />
fixed Boolean false Sets icon fixed option <mdb-icon fixed />
pull [Boolean, String] false Sets icon pull option <mdb-icon pull />
border Boolean false Sets icon border <mdb-icon border />
spin Boolean false Sets icon spin option <mdb-icon spin />
pulse Boolean false Sets icon pulse option <mdb-icon pulse />
rotate [Boolean, String] false Sets icon rotate option <mdb-icon rotate />
flip [Boolean, String] false Sets icon flip option <mdb-icon flip />
inverse [Boolean, String] false Sets icon inverse option <mdb-icon inverse />
stack [Boolean, String] false Sets icon stack option <mdb-icon stack />
color String '' Sets icon color <mdb-icon color="primary" />
customIconClass [String, Array] '' Sets icon custom class <mdb-icon customIconClass="text-success" />
