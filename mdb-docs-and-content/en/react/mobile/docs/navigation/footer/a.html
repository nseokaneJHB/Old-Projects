Footer - API

In this section you will find advanced information about the Footer component. You will find out which modules are required, what are the possibilities of configuring the component, and what events and methods you can use in working with it.

Import statement

import { MDBFooter, MDBFooterCopyright} from 'mdbreact-mobile'

API Reference: Properties

Name Type Default Description Example
color string #fff Color API you can use color from our pallete or set color what you like in HEX, rgb/a or HSL. <MDBFooter color="#fff" />
textColor string #000 TextColor API you can use color from our pallete or set color what you like in HEX, rgb/a or HSL. <MDBFooter textColor="rgb(0,0,0)" />
style object Set any styles to footer. <MDBFooter style={{fontSize: 16}} />
children default slot projection #fff Set any component to this props to display. <MDBFooter />children</MDBFooter>
