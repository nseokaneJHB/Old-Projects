React Bootstrap - Color Picker

React Color Picker - Bootstrap 4 & Material Design

React Bootstrap Color Picker plugin allows you to select different colors. You can successfully use it in various product wizards, clothing sales, or other situations where you need to be able to choose a color.

To see the list of all properties and methods of the plugin see "API" tab on this page.
