Bootstrap table responsive

Responsive tables allow tables to be scrolled horizontally with ease. Make any table responsive across all viewports by adding responsive prop to Table component. Or, pick a maximum breakpoint with which to have a responsive table up to by using responsive{Sm|Md|Lg|Xl}.

Always responsive

Across every breakpoint, use responsive prop for horizontally scrolling tables.

Breakpoint specific

Use responsive{Sm|Md|Lg|Xl} prop as needed to create responsive tables up to a particular breakpoint. From that breakpoint and up, the table will behave normally and not scroll horizontally.
