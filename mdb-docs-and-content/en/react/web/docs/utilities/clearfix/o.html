React Clearfix - Bootstrap 4 & Material Design

React Bootstrap clearfix is a utility which to automatically clear child elements, so a user doesn't have to add additional markup.

Quickly and easily clear floated content by adding a clearfix utility ( .clearfix) to the parent element. Can also be used as a mixin.
