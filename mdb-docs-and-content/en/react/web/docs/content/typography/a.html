React Typography - API

In this section you will find advanced information about the Typography component. You will find out which modules are required, what are the possibilities of configuring the component, and what events and methods you can use in working with it.

Typography import statement

In order to use Typography component make sure you have imported proper module first.

import { MDBTypography } from "mdbreact"; //or import { MDBTypo } from "mdbreact";

API Reference: Typography Properties

The table below shows the configuration options of the MDBTypography component.

Name Type Default Description Example
className string Adds custom classes <MDBTypography className="myClass" />
tag string p Sets color of the Typography. Choose one of these: 'ul', 'p', 'h1', 'h2', 'h3', 'h4', 'h5', 'h6' <MDBTypography tag="h1" />
variant string Set a text variant to choosen tag. <MDBTypography variant="h1" />
blockquote boolean false Set blockqoute to tag. <MDBTypography blockquote />
bgColor string Sets color of the Typography. Choose one of these: 'primary', 'default', 'success', 'warning', 'danger', 'info', 'secondary', 'light' <MDBTypography bgColor="primary" />
bgTitle string Set title to blockquote. <MDBTypography bqTitle="Your title" />
listUnStyled boolean false Set ul list with un-styled class. <MDBTypography listUnStyled />
listInLine boolean false Set ul list with list-in-line class and set elemnt in line. <MDBTypography listUnStyled />
colorText string Sets color of the Typography. Choose one of these: 'red','pink','purple' ,'deep-purple','indigo','blue','light-blue','cyan','teal','green' ,'light-green','lime','yellow','amber','orange','deep-orange','brown' ,'grey','blue-grey','mdb-color white'. <MDBTypography colorText="pink" />
note boolean false Set to true if you want to have a note. <MDBTypography note/>
noteColor string Sets color of the Note Typography. Choose one of these: 'primary', 'default', 'success', 'warning', 'danger', 'info', 'secondary', 'light', 'dark', 'unique', 'stylish', 'special', 'elegant'. <MDBTypography noteColor="danger/>
