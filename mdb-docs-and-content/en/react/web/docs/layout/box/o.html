React Bootstrap Box

React Box - Bootstrap 4 & Material Design

React Bootstrap boxes are layout components which help to write build components for of the CSS utility needs.

Basic examples

Box component can be any component from React.ElementType like div, span, img, video etc. The basic component is a div.

Div
Span

Paragraph

Section
import React from "react"; import { MDBBox } from 'mdbreact'; const BoxPage = () => { return ( Div Span Paragraph Section ); }; export default BoxPage;

Display

With MDBBox component you can set any display CSS style from Bootstrap 4 to MDBBox component like d-none, d-flex etc.

You can read more about display here.

d-flex
d-block
d-none
d-inline
import React from "react"; import { MDBBox } from 'mdbreact'; const BoxPage = () => { return ( d-flex d-block d-none d-inline ); }; export default BoxPage;

Flex

With MDBBox component you can set flexbox CSS styles to this component and render it on your website. You can use props justifyContent, flex, alignItems, alignContent, alignSelf.

You can read more about flex here.

d-flex justify-content-center
d-flex justify-content-end

Colors

With MDBBox component you can set text CSS colors to element between MDBBox component.

You can read more about colors here.

red color
pink color
blue color
cyan color
import React from "react"; import { MDBBox } from 'mdbreact'; const BoxPage = () => { return ( red color pink color blue color cyan color ); }; export default BoxPage;
