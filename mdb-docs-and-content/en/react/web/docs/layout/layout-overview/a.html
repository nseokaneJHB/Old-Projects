React Container - API

In this section you will find advanced information about the Container component. You will find out which modules are required, what are the possibilities of configuring the component, and what events and methods you can use in working with it.

Container import statement

In order to use Box component make sure you have imported proper module first.

import { MDBContainer } from "mdbreact";

API Reference: Container Properties

The table below shows the configuration options of the MDBContainer component.

Name Type Default Description Example
className String Adds custom classes <MDBContainer className="myClass" />
tag String div Changes default tag <MDBContainer tag="section" />
fluid boolean false Add container-fluid class <MDBContainer fluid />
size string This props allows to use responsive containers. You can set container with breakpoints like sm, md, lg, xl. <MDBContainer size="sm" />
